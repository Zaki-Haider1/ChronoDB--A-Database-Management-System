BASIC OPERATIONS 

1. CREATE TABLE
   Syntax: CREATE TABLE <table_name> (<field1> <type>, <field2> <type>, ...);
   Example: CREATE TABLE students (id INT, name STRING, gpa FLOAT);
   
2. INSERT
   Syntax: INSERT INTO <table_name> VALUES <id> <name> <gpa>;
   Example: INSERT INTO students VALUES 1 Alice 3.8;
   Example: INSERT INTO students VALUES 2 Bob 3.5;

3. SELECT
   Syntax: SELECT * FROM <table_name>;
   Example: SELECT * FROM students;
   Note: Shows all records in formatted table

4. UPDATE
   Syntax: UPDATE <table_name> SET <field> <value> WHERE ID <id>;
   Example: UPDATE students SET gpa 4.0 WHERE ID 1;
   Example: UPDATE students SET name Charlie WHERE ID 2;
   Valid Fields: NAME, GPA

5. DELETE
   Syntax: DELETE FROM <table_name> WHERE ID <id>;
   Example: DELETE FROM students WHERE ID 2;
   Note: Removes record by ID

6. UNDO
   Syntax: UNDO;
   Example: UNDO;
   Note: Reverts the last operation (CREATE, INSERT, UPDATE, DELETE)

7. REDO
   Syntax: REDO;
   Example: REDO;
   Note: Re-applies the last undone operation

8. EXIT
   Syntax: EXIT; (or exit; - case insensitive)
   Example: exit;
   Note: Closes the ChronoDB CLI


GRAPH OPERATIONS

1. CREATE GRAPH
   Syntax: GRAPH CREATE <graph_name>;
   Example: GRAPH CREATE myGraph;
   Note: Creates a new graph structure

2. IMPORT DATA (NEW)
   Syntax: GRAPH IMPORT <graph_name> FROM <table_name> COLUMN <col_name>;
   Example: GRAPH IMPORT myGraph FROM students COLUMN name;
   Note: Automatically creates vertices from a table column

3. SHOW GRAPH (NEW)
   Syntax: GRAPH SHOW <graph_name>;
   Example: GRAPH SHOW myGraph;
   Note: Opens the 2D Visualization Window

4. ADD EDGE
   Syntax: GRAPH ADDEDGE <graph_name> <vertex1> <vertex2> <weight>;
   Example: GRAPH ADDEDGE myGraph A B 5;
   Note: Creates connection between two vertices with weight (Vertices must exist or be imported)

5. PRINT GRAPH
   Syntax: GRAPH PRINT <graph_name>;
   Example: GRAPH PRINT myGraph;
   Note: Displays the adjacency list in console

6. ALGORITHMS
   Syntax: GRAPH BFS <graph_name> <start>;
   Syntax: GRAPH DFS <graph_name> <start>;
   Syntax: GRAPH DIJKSTRA <graph_name> <start> <end>;
   Example: GRAPH DIJKSTRA myGraph A B;


EXAMPLE WORKFLOW

ChronoDB> CREATE TABLE cities (name STRING, pop INT);
[SUCCESS]: Table 'cities' created.

ChronoDB> INSERT INTO cities VALUES Lahore 1000;
[SUCCESS]: Record inserted.

ChronoDB> INSERT INTO cities VALUES Karachi 2000;
[SUCCESS]: Record inserted.

ChronoDB> GRAPH CREATE Map;
[SUCCESS]: Graph 'Map' created successfully.

ChronoDB> GRAPH IMPORT Map FROM cities COLUMN name;
[SUCCESS]: Imported 2 nodes into Map.

ChronoDB> GRAPH ADDEDGE Map Lahore Karachi 1200;
[SUCCESS]: Edge added.

ChronoDB> GRAPH SHOW Map;
(Opens Visual Window)

ChronoDB> EXIT;
